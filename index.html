Dronesoccer
- Each team consists of five players. Two teams maneuver their own drone ball, driving it toward the opponent’s goal post while defending their own. Only the striker from each team can score goals.
- A Dronesoccer game consists of three sets. Each set lasts 3 minutes, and number of goals scored during each set determines the winner and loser of the game.
